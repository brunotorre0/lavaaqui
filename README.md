# LavaAqui

Aplicação híbrida para gestão de lavandarias desenvolvida com Ionic, Angular e Capacitor.

## Pré-requisitos

- Node.js (v18 ou superior)
- npm ou yarn
- Para Android: Android Studio
- Para iOS: Xcode (apenas no macOS)

## Instalação

```bash
npm install
```

## Executar

```bash
npm start
```

A aplicação estará disponível em `http://localhost:4200`.

## Build para Produção

```bash
npm run build:prod
```

## Scripts Disponíveis

- `npm start` - Inicia servidor de desenvolvimento
- `npm run build` - Build de desenvolvimento
- `npm run build:prod` - Build de produção
- `npm run android` - Build + Sync + Abre Android Studio
- `npm run ios` - Build + Sync + Abre Xcode

## Tecnologias

- Ionic 8
- Angular 20
- Capacitor 7
